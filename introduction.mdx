## What can the Igbo API do?

The Igbo API is a low-level API that focuses on making it easier for developers to request Igbo words data.

Having the ability to access our database of 25,000+ Igbo words and 25,000+ Igbo sentence enables you to create some interesting process.

Just to name a few:

- Dictionary app
- Chrome extensions
- Igbo language learning app
